# kaoyan-vocab-html
# 考研单词中文同义归类背诵
一个适合手机端使用的考研英语词汇离线背诵 HTML 页面。  
支持按照中文义项、常用大类、A-Z 单词、补充释义进行分类浏览，并提供收藏、已掌握、随机背诵、进度导入导出等功能。
## 功能
- 中文同义 / 近义分类背诵
- 常用大类分类
- A-Z 全部单词索引
- 单词收藏与已掌握标记
- 分类收藏与已掌握标记
- 随机背 20 个
- 学习进度导出 / 导入
- 单 HTML 离线使用，适合手机浏览器打开
## 使用方式
下载 HTML 文件后，直接用浏览器打开即可使用。  
## 进度备份
页面内支持导出学习进度为 JSON 文件。  
更换浏览器、手机或清理缓存前，建议先导出进度备份。

本项目主要用于个人考研英语词汇复习和离线背诵工具展示,代码纯由Codex生成
